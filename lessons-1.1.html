УРОК 1 - Создание персонажа

В Content Browser перейдите к папке Blueprints. Нажмите на кнопку Add New и выберите Blueprint Class.

Рисунок 1 - Добавление класса


Выберите во всплывающем окне Character. Character Наследуеться от Pawn но этот класс содержит зарание заготовки свойственные человеку. Назовите его BP_Player.

Рисунок 2 - Создание класса


Далее левой кнопкой мыши перетаскиваем из Content Browser на игровую арену во viewport. На вкладку World Outliner или же в сомом viewport выбираем класс BP_Player и переходим во вкладку Details Находим пункт Auto Possess Player и выбираем Player 0

Рисунок 3 - Свойства объектов


Это закрепит играка в классе Character при старте игры. Также можно использовать другой способ подключении играка через GameMode. Для этого добовляем новый Blueprint class. Делаем анологичные действия добовление класса и выбираем на этот раз класс Game Mode Base и называем его gamemode. После этого в Content Browser открываем его и в Default pawn Class указываем класс BP_Player.

Рисунок 4 - Класс gamemode


Сам же класс GameMode необходимо закрепить в настройках проекта выбрав вкладку Project Settings в проекте или же открыть ее из меню Edit. Тут выбираем настойки проекта Project>>Maps & Modes Устанавлеваем наш созданный геймод и карту по умолчанию.

Рисунок 5 - Начальные настройки


Этот способ евляется более правильным и привязывает игрока ко всем уровням игры.
